OVERVIEW

This module steps free surface height forward in time using a leap-frog time stepping scheme.

PUBLIC ROUTINES

  1. ocean_freesurf_init

    DESCRIPTION
    Initialization routine


  2. update_ocean_freesurf

    DESCRIPTION

  3. ocean_freesurf_end
